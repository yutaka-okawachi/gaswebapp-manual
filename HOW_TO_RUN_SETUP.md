# setupGitHubCredentials() の実行方法（簡易版）

## 🎯 実行場所

**Apps Scriptエディタ**で実行します（スプレッドシートではありません）

---

## 📋 3ステップで完了

### 1️⃣ Apps Scriptエディタを開く

スプレッドシート → **拡張機能** → **Apps Script**

### 2️⃣ 関数を選択

GASエディタの**上部ツールバー**:

![GAS実行方法](C:/Users/okawa/.gemini/antigravity/brain/6cd43a9a-2937-47cb-8548-c613ad2b48d4/gas_execution_steps_1764823481169.png)

1. 「関数を選択」ドロップダウンをクリック
2. **`setupGitHubCredentials`** を選択
3. 右側の **実行** ボタン（▶）をクリック

---

### 3️⃣ 権限を許可（初回のみ）

初回実行時のみ、権限リクエストが表示されます：

1. **権限を確認** → 自分のアカウント選択
2. 「このアプリは確認されていません」→ **詳細** → **移動**
3. **許可** をクリック

---

## 💬 UIプロンプトで入力

スプレッドシートにダイアログが4回表示されます：

| ステップ | 入力内容 |
|---------|---------|
| 1/4 | GitHub Personal Access Token（作成したトークン） |
| 2/4 | GitHubユーザー名（`yutaka-okawachi`） |
| 3/4 | リポジトリ名（`gaswebapp-manual`） |
| 4/4 | ブランチ名（`main`） |

各ステップで **OK** をクリック

---

## ✅ 完了確認

「✓ 設定完了」メッセージが表示されれば成功！
