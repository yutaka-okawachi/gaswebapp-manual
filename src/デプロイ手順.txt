### B) 手動でデプロイする方法

# 1. プロジェクトフォルダに移動
cd "C:\Users\okawa\gaswebapp-manual"

# 2. ローカルの変更を反映
clasp push

# 3. 新しいバージョンを作成し、番号を控える (例: 567)
clasp version "検索ボタンのUIを修正"

# 4. 控えたバージョン番号でデプロイ
clasp deploy -i AKfycbxWl-KLwo8SnOyqQT84gJyrofRQnIp_GBv8Pg0N5athPAoxp9LBuwj0HDTXkFqh0xiGsw -V 567 -d "検索ボタンのUIを修正"
