# マニュアル一覧

このディレクトリには、プロジェクトの運用・保守に関する集約マニュアルが含まれています。
秘密情報そのものは書き込まないでください。

---

## 1. [プロジェクト引き継ぎ・長期保守ガイド (SUCCESSOR_GUIDE.md)](SUCCESSOR_GUIDE.md)
* **対象者**: プロジェクトの前任者、および引き継ぎを受ける後継者（PC操作に詳しくない初心者も含みます）
* **内容**:
  * 資産一覧
  * 初回の対話式セットアップ画面の答え方・動作フロー
  * アカウント共有・権限引き継ぎ時のチェックリスト
  * 長期的なアカウント無効化・Successor（継承者）の登録設定
  * 技術的な注意事項（サイドバーの二重管理など）

## 2. [開発者・運用保守ガイド (DEVELOPER_GUIDE.md)](DEVELOPER_GUIDE.md)
* **対象者**: 開発者、およびコマンドライン操作に慣れている保守管理者
* **内容**:
  * ローカル開発環境の構築（Node.js, Git, clasp, `.env` 設定）
  * 同期スクリプト (`sync-data.ps1`) の動作フローとトラブルシューティング（GASの200バージョン制限対策やGit競合対策など）
  * 楽譜メタデータの修正・同期方法
  * サイトのSEO（canonical設定、sitemap.xmlの自動更新）およびパフォーマンス最適化方針
