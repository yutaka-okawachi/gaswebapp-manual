# clasp認証手順

## 現在の状況

`clasp login`コマンドを実行中です。ブラウザが開いて認証画面が表示されているはずです。

## 手順

### 1. ブラウザを確認

新しいタブまたはウィンドウが開いているはずです。開いていない場合は、ターミナルに表示されているURLをブラウザに貼り付けてください。

### 2. Googleアカウントでログイン

GASプロジェクトにアクセスできるGoogleアカウントを選択してログイン

### 3. 権限を許可

- 「claspがGoogleアカウントにアクセスすることを許可しますか？」
- **「許可」をクリック**

### 4. 完了を確認

- ブラウザに「認証に成功しました」のようなメッセージが表示されます
- ターミナルに戻って `Authorization successful.` と表示されます

## 認証完了後

認証が完了したら、以下のコマンドでclasp pushをテストします：

```bash
clasp push
```

成功すれば、GASプロジェクトが自動的に更新されるようになります。

## トラブルシューティング

**Q: ブラウザが開かない**
A: ターミナルに表示されているURLを手動でコピーしてブラウザに貼り付けてください

**Q: 権限エラーが出る**
A: GASプロジェクトにアクセスできるGoogleアカウントを使用していることを確認してください
